About this site

This is a student-built demo for the FBLA Website Coding & Development event. It emphasizes accessibility, performance, and privacy-friendly design.

  • Accessible: keyboard-friendly, ARIA labels, strong color contrast.
  • Fast: optimized images, lazy loading, minimal JS, offline-ready.
  • Privacy-first: no personal data stored on a server in this demo.
